# Employee Management System

A Spring Boot REST API project for managing employee records using Java, Spring Boot, Spring Data JPA, Hibernate, and MySQL.

---

## 🚀 Tech Stack

- Java 25
- Spring Boot 4
- Spring Data JPA
- Hibernate
- MySQL
- Maven
- IntelliJ IDEA
- Git & GitHub
